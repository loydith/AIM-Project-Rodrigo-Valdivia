The Wuw

My name is Rodrigo Valdivia

Currently I'm in fourth grade and this project is created for the class AIM. I desired to make this project, because most kids have a problem to wake up in the morning early for school.

Description of the Wuw

The Wuw is an alarm clock that sprays water on you when you hit either the snooze or the off button. The Wuw comes in different colors such as white, blue, red and purple.
The Wuw always makes sure that you wake up, so it is better to buy the Wus than just any other alarm clock.
